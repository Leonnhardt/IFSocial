Tecnologias:

Javascript

Javascript é uma linguagem de programação utilizada para que as respostas aos comandos do usuário possam ser processadas com mais velocidade. Além disso,  tivemos uma boa introduçao ao javascript no segundo módulo do curso.

HTML/CSS

Para a parte de conteúdo visual das páginas, usaremos HTML como linguagem de marcação e o CSS para a parte de estilos, pois tivemos várias aulas de como construir páginas usando essas tecnologias.
